# 1. Set up AWS account structure:

• Follow the guidance in the Organizing Your AWS Environment Using Multiple Accounts
whitepaper to create separate AWS accounts for different environments (for exampoe,
development, staging, and production).
• Configure appropriate access controls and permissions for each account to restrict direct
human access to production environments.
