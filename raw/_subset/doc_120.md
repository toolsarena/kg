# 8. Review AWS Resource Access Manager Shares periodically:

• You can use AWS Resource Access Manager to share resources, such as AWS Network Firewall
policies, Amazon Route 53 resolver rules, and subnets, within your Amazon VPCs.
• Audit shared resources regularly and stop sharing resources that no longer need to be shared.
