# Implementation guidance

Maintain assets in version controlled repositories. Doing so supports tracking changes, deploying
new versions, detecting changes to existing versions, and reverting to prior versions (for example,
rolling back to a known good state in the event of a failure). Integrate the version control
capabilities of your configuration management systems into your procedures.
