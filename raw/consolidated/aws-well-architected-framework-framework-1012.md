# AWS Well-Architected Framework Framework

If the third party no longer requires access to your environment, remove the role. Avoid providing
long-term credentials to a third party. Maintain awareness of other AWS services that support
sharing, such as the AWS Well-Architected Tool allowing sharing a workload with other AWS
accounts, and AWS Resource Access Manager helping you securely share an AWS resource you own
with other accounts.
