# AWS Well-Architected Framework Framework

AWS CodeDeploy automates software deployments to a variety of compute services, such as
Amazon EC2, AWS Fargate, AWS Lambda, and your on-premise servers. To configure these steps,
see Getting started with CodeDeploy.
• Set up a continuous delivery service that automates your release pipelines for quicker and
more reliable application and infrastructure updates: Consider using AWS CodePipeline to help
you automate your release pipelines. For more detail, see CodePipeline tutorials.
