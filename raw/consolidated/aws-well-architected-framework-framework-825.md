# AWS Well-Architected Framework Framework

Over time, these procedures should be evolved to be run as code, reducing the need for human
intervention. For example, procedures can be initiated as AWS Lambda functions, CloudFormation
templates, or AWS Systems Manager Automation documents. Verify that these procedures are
version-controlled in appropriate repositories, and include suitable resource tagging so that teams
can readily identify owners and documentation. Document the responsibility for carrying out the
activities, and then monitor the automations for successful initiation and operation, as well as
performance of the desired outcomes.
