# AWS Well-Architected Framework Framework

• You introduce architecture changes over time without justification.
Benefits of establishing this best practice: By having a well-defined approach for making
architectural choices, you use data to influence your workload design and make informed decisions
over time.
Level of risk exposed if this best practice is not established: Medium
