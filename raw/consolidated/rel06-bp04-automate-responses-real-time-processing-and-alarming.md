# REL06-BP04 Automate responses (Real-time processing and alarming)

Use automation to take action when an event is detected, for example, to replace failed
components.
Automated real-time processing of alarms is implemented so that systems can take quick
corrective action and attempt to prevent failures or degraded service when alarms are triggered.
