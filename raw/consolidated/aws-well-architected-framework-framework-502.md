# AWS Well-Architected Framework Framework

• Composite alarms can account for complex alarms from multiple sources.
• Once the alarm has been created, create appropriate notification events. You can directly
invoke an Amazon SNS API to send notifications and link any automation for remediation or
communication.
• Stay informed about service degradations with AWS Health. Create purpose-fit AWS Health
event notifications to e-mail and chat channels through AWS User Notifications and integrate
programmatically with your monitoring and alerting tools through Amazon EventBridge.


# AWS Well-Architected Framework Framework

Desired outcome: Each application has a defined target for availability and SLA for performance
metrics, which can be monitored and maintained in order to meet business outcomes.
