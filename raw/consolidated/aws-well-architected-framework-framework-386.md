# AWS Well-Architected Framework Framework

Benefits of establishing this best practice: Active planning of your network will ensure that you
do not have multiple occurrences of the same IP address in interconnected networks. This prevents
routing problems from occurring in parts of the workload that are using the different applications.
Level of risk exposed if this best practice is not established: Medium
