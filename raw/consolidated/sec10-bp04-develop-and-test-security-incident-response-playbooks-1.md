# SEC10-BP04 Develop and test security incident response playbooks

A key part of preparing your incident response processes is developing playbooks. Incident
response playbooks provide prescriptive guidance and steps to follow when a security event occurs.
Having clear structure and steps simplifies the response and reduces the likelihood for human
error.
Level of risk exposed if this best practice is not established: Medium
