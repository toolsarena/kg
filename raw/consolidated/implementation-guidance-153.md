# Implementation guidance

• CloudWatch Logs Insights allows you to interactively search and analyze your log data in Amazon
CloudWatch Logs.
• Analyzing Log Data with CloudWatch Logs Insights
• Amazon CloudWatch Logs Insights Sample Queries
• Use Amazon CloudWatch Logs to send logs to Amazon S3 where you can use or Amazon Athena
to query the data.
• How do I analyze my Amazon S3 server access logs using Athena?
• Create an S3 lifecycle policy for your server access logs bucket. Configure the lifecycle policy
to periodically remove log files. Doing so reduces the amount of data that Athena analyzes
for each query.
