# AWS Well-Architected Framework Framework

• To support continual improvement, you approve administrators and developers using all their
extra time to select and implement improvements. No improvements are ever completed.
• Operational acceptance is complete, and you do not test operational practices again.
Benefits of establishing this best practice: By dedicating time and resources within your
processes, you can make continuous, incremental improvements possible.
Level of risk exposed if this best practice is not established: Low
