# AWS Well-Architected Framework Framework

Desired outcome: You define network protections with templates and commit them into a version
control system. Automated pipelines are initiated when new changes are made that orchestrates
their testing and deployment. Policy checks and other static tests are in place to validate changes
before deployment. You deploy changes into a staging environment to validate the controls
are operating as expected. Deployment into your production environments is also performed
automatically once controls are approved.
