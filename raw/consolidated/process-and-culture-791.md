# Process and culture 791

| Service | Where to use |
| --- | --- |
| Amazon CloudWatch Real-User Monitoring
(RUM) | To capture application performance metrics
from real user client-side and frontend
sessions. |
| AWS X-Ray | To trace traffic through the application layers
and identify latency between components
and dependencies. Use X-Ray service maps
to see relationships and latency between
workload components. |
| Amazon Relational Database Service
Performance Insights | To view database performance metrics and
identify performance improvements. |
| Amazon RDS Enhanced Monitoring | To view database OS performance metrics. |
| Amazon DevOps Guru | To detect abnormal operating patterns so
you can identify operational issues before
they impact your customers. |
