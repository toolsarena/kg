# Amazon CloudWatch RUM Use Amazon CloudWatch RUM to collect the

metrics that give you the insights that help
you identify, understand, and improve user
experience.
• Identify top talkers and application traffic patterns using VPC and AWS Transit Gateway Flow
Logs.
• Assess and optimize your current network architecture including VPCs, subnets, and routing. As
an example, you can evaluate how different VPC peering or AWS Transit Gateway can help you
improve the networking in your architecture.
• Assess the routing paths in your network to verify that the shortest path between destinations is
always used. Network Access Analyzer can help you do this.
