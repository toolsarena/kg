# Implementation steps

• Understand data characteristics: Analyze your data characteristics and access patterns to
identify the correct configuration for your cloud resources. Key characteristics to consider
include:
• Data type: structured, semi-structured, unstructured
• Data growth: bounded, unbounded
• Data durability: persistent, ephemeral, transient
