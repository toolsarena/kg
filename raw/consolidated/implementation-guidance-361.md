# Implementation guidance

• Allocate time to make improvements: Dedicate time and resources within your processes to make
continuous, incremental improvements.
• Implement changes to improve and evaluate the results to determine success.
• If the results do not satisfy the goals and the improvement is still a priority, pursue alternative
courses of action.
• Simulate production workloads through game days, and use learnings from these simulations to
improve.
