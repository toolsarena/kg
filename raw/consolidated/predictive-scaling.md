# Predictive scaling

Predictive scaling uses machine learning to automatically scale resources based on anticipated
demand. Predictive scaling analyzes the historical value of a utilization metric you provide and
continuously predicts its future value. The predicted value is then used to scale the resource up or
down. Amazon EC2 Auto Scaling can perform predictive scaling.
