# Networking and content delivery 785

| AWS tool | Where to use |
| --- | --- |
| AWS Network Manager | Using AWS Network Manager, you can
monitor the real-time and historical
performance of the AWS Global Network for
operational and planning purposes. Network
Manager provides aggregate network latency
between AWS Regions and Availability Zones
and within each Availability Zone, allowing
you to better understand how your applicati
on performance relates to the performance
of the underlying AWS network. |
| Amazon CloudWatch RUM | Use Amazon CloudWatch RUM to collect the
metrics that give you the insights that help
you identify, understand, and improve user
experience. |
