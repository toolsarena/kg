# Implementation guidance

Implement practices to improve code quality to minimize defects before they are deployed. Use
practices like test-driven development, code reviews, and pair programming to increase the quality
of your development.
Use the power of generative AI with Amazon Q Developer to improve developer productivity
and code quality. Amazon Q Developer includes generation of code suggestions (based on large
language models), production of unit tests (including boundary conditions), and code security
enhancements through detection and remediation of security vulnerabilities.


# Implementation guidance

Use frequent, small, and reversible changes to reduce the scope and impact of a change. This eases
troubleshooting, helps with faster remediation, and provides the option to roll back a change. It
also increases the rate at which you can deliver value to the business.
