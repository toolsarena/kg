# AWS Well-Architected Framework Framework

1. Use a version control system to store and manage assets (such as documents, source code, and
binary files).
2. Use CodeBuild to compile your source code, runs unit tests, and produces artifacts that are ready
to deploy.
3. Use CodeDeploy as a deployment service that automates application deployments to Amazon
EC2 instances, on-premises instances, serverless AWS Lambda functions, or Amazon ECS.
4. Monitor your deployments.
