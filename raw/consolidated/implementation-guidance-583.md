# Implementation guidance

When you remove unneeded and redundant datasets, you can reduce storage cost and
environmental footprint. This practice may also make computing more efficient, as compute
resources only process important data instead of unneeded data. Automate the deletion of
unneeded data. Use technologies that deduplicate data at the file and block level. Use service
features for native data replication and redundancy.
