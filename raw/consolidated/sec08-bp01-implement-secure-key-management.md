# SEC08-BP01 Implement secure key management

Secure key management includes the storage, rotation, access control, and monitoring of key
material required to secure data at rest for your workload.
Desired outcome: You have a scalable, repeatable, and automated key management mechanism.
The mechanism enforces least privilege access to key material and provides the correct balance
between key availability, confidentiality, and integrity. You monitor access to the keys, and if
rotation of key material is required, you rotate them using an automated process. You do not allow
key material to be accessed by human operators.
