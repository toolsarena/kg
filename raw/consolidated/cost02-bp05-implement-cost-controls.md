# COST02-BP05 Implement cost controls

Implement controls based on organization policies and defined groups and roles. These certify that
costs are only incurred as defined by organization requirements such as control access to regions or
resource types.
Level of risk exposed if this best practice is not established: Medium


# COST02-BP06 Track project lifecycle

Track, measure, and audit the lifecycle of projects, teams, and environments to avoid using and
paying for unnecessary resources.
