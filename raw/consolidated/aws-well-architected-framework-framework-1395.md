# AWS Well-Architected Framework Framework

• Implement controls on usage: Using your defined organization policies, implement IAM policies
and roles to specify which actions users can perform and which actions they cannot. Multiple
organizational policies may be included in an AWS policy. In the same way that you defined
policies, start broadly and then apply more granular controls at each step. Service limits are also
an effective control on usage. Implement the correct service limits on all your accounts.
