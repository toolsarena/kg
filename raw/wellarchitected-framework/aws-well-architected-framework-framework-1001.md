# AWS Well-Architected Framework Framework

Where supported, resource policies define the conditions that a principal must satisfy to gain
access to a resource. Resource policies also scope down the set of allowable actions, where
appropriate. Permission boundaries are placed on principals that manage workload permissions,
delegating permission management to individual workload owners.
