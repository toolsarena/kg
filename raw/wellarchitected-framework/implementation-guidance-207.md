# Implementation guidance

AWS offers services like AWS Global Accelerator and Amazon CloudFront that can help improve
network performance, while most AWS services have product features (such as the Amazon S3
Transfer Acceleration feature) to optimize network traffic.
Review which network-related configuration options are available to you and how they could
impact your workload. Performance optimization depends on understanding how these options
interact with your architecture and the impact that they will have on both measured performance
and user experience.
