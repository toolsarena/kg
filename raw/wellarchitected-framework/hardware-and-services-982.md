# Hardware and services 982

| Task | What to use on AWS |
| --- | --- |
| Hosting a database | Use managed Amazon Relational Database
Service (Amazon RDS) instances instead of
maintaining your own Amazon RDS instances
on Amazon Elastic Compute Cloud (Amazon
EC2). |
| Hosting a container workload | Use AWS Fargate, instead of implementing
your own container infrastructure. |
| Hosting web apps | Use AWS Amplify Hosting as fully managed
CI/CD and hosting service for static websites
and server-side rendered web apps. |
