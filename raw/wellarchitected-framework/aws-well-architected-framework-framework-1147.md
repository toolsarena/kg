# AWS Well-Architected Framework Framework

you integrate with directly native AWS services and other web services. API Gateway supports
the OpenAPI specification and versioning. AWS AppSync is a managed GraphQL endpoint you
configure by defining a GraphQL schema to define a service interface for queries, mutations
and subscriptions. Amazon EventBridge uses event schemas to define events and generate code
bindings for your events.
