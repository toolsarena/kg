# AWS Well-Architected Framework Framework

Benefits of establishing this best practice: Proactive remediation of alarm actions allows support
staff to concentrate on those items that are not automatically actionable. This helps operations
staff handle all alarms without being overwhelmed and instead focus only on critical alarms.
Level of risk exposed if this best practice is not established: Low
