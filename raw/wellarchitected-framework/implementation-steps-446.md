# Implementation steps

• Allocate costs to workload metrics: Using the defined metrics and configured tags, create a
metric that combines the workload output and workload cost. Use analytics services such as
Amazon Athena and Amazon Quick to create an efficiency dashboard for the overall workload
and any components.
