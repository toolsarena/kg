# Implementation steps

1. Set up escalation prompts: Set up CloudWatch alarms to create an incident in AWS Systems
Manager Incident Manager.
2. Set up on-call schedules: Create on-call schedules in Incident Manager that align with your
escalation paths. Equip on-call personnel with the necessary permissions and tools to act swiftly.
