# AWS Well-Architected Framework Framework

• Select a managed device farm: Select a managed device farm that can support your testing
requirements. For example, you can use AWS Device Farm to test and understand the impact of
your changes on a representative set of hardware.
• Use automation: Use automation and continuous integration/continuous deployment (CI/CD) to
schedule and run your tests.
• Integrating AWS Device Farm with your CI/CD pipeline to run cross-browser Selenium tests
• Building and testing iOS and iPadOS apps with AWS DevOps and mobile services
• Review and adjust: Continually review your testing results and make necessary improvements.
