# SEC05-BP04 Automate network protection

Automate the deployment of your network protections using DevOps practices, such as
infrastructure as code (IaC) and CI/CD pipelines. These practices can help you track changes in your
network protections through a version control system, reduce the time it takes to deploy changes,
and help detect if your network protections drift from your desired configuration.
