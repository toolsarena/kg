# COST08-BP03 Implement services to reduce data transfer costs

Implement services to reduce data transfer. For example, use edge locations or content delivery
networks (CDN) to deliver content to end users, build caching layers in front of your application
servers or databases, and use dedicated network connections instead of VPN for connectivity to
the cloud.
Level of risk exposed if this best practice is not established: Medium
