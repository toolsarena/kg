# Common anti-patterns:

• There isn't clear guidance or mechanisms in an organization for what to do when a risk is
identified. For example, when an employee notices a phishing attack, they fail to report to the
security team, resulting in a large portion of the organization falling for the attack. This causes a
data breach.
