# Implementation guidance

Architecting for data transfer minimizes data transfer costs. This may involve using content
delivery networks to locate data closer to users, or using dedicated network links from your
premises to AWS. You can also use WAN optimization and application optimization to reduce the
amount of data that is transferred between components.
When transferring data to or within the AWS Cloud, it is essential to know the destination based on
varied use cases, the nature of the data, and the available network resources in order to select the
right AWS services to optimize data transfer. AWS offers a range of data transfer services tailored
for diverse data migration requirements. Select the right data storage and data transfer options
based on the business needs within your organization.
