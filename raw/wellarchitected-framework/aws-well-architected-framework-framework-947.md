# AWS Well-Architected Framework Framework

• You use version-controlled repositories to share application libraries, scripted procedures,
procedure documentation, and other system documentation.
• You share your infrastructure standards as version-controlled AWS CloudFormation templates.
• You review lessons learned across teams.
