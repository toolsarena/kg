# Implementation steps

• Evaluate data and access characteristics: Evaluate your data characteristics and access pattern
to collect the key characteristics of your storage needs. Key characteristics to consider include:
• Data type: structured, semi-structured, unstructured
• Data growth: bounded, unbounded
• Data durability: persistent, ephemeral, transient
• Access patterns: reads or writes, frequency, spiky, or consistent
