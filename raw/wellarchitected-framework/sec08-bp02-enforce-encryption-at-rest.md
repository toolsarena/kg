# SEC08-BP02 Enforce encryption at rest

Encrypt private data at rest to maintain confidentiality and provide an additional layer of
protection against unintended data disclosure or exfiltration. Encryption protects data so that it
cannot be read or accessed without first being decrypted. Inventory and control unencrypted data
to mitigate risks associated with data exposure.
Desired outcome: You have mechanisms that encrypt private data by default when at rest.
