# Implementation guidance

To improve long-term workload sustainability, use architecture patterns that support data access
and storage characteristics for your workload. These patterns help you efficiently retrieve and
process data. For example, you can use modern data architecture on AWS with purpose-built
services optimized for your unique analytics use cases. These architecture patterns allow for
efficient data processing and reduce the resource usage.
