# AWS Well-Architected Framework Framework

Desired outcome: You build your workload from external software packages in addition to the
code that you write. This makes it simpler for you to implement functionality that is repeatedly
used, such as a JSON parser or an encryption library. You centralize the sources for these packages
and dependencies so your security team can validate them before they are used. You use this
approach in conjunction with the manual and automated testing flows to increase the confidence
in the quality of the software that you develop.
