# AWS Well-Architected Framework Framework

• OPS02-BP01 Resources have identified owners - Resources need identified owners before you
build a change management process.
• OPS02-BP02 Processes and procedures have identified owners - Processes need identified
owners before you build a change management process.
• OPS02-BP03 Operations activities have identified owners responsible for their performance -
Operations activities need identified owners before you build a change management process.
