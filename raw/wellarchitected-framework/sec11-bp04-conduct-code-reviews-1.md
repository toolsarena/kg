# SEC11-BP04 Conduct code reviews

Implement code reviews to help verify the quality and security of software being developed. Code
reviews involve having team members other than the original code author review the code for
potential issues, vulnerabilities, and adherence to coding standards and best practices. This process
helps catch errors, inconsistencies, and security flaws that might have been overlooked by the
original developer. Use automated tools to assist with code reviews.
