# AWS Well-Architected Framework Framework

PERF03-BP01 Use a purpose-built data store that best supports your data access and storage
requirements
Understand data characteristics (like shareable, size, cache size, access patterns, latency,
throughput, and persistence of data) to select the right purpose-built data stores (storage or
database) for your workload.
