# COST05-BP06 Perform cost analysis for different usage over time

Workloads can change over time. Some services or features are more cost effective at different
usage levels. By performing the analysis on each component over time and at projected usage, the
workload remains cost-effective over its lifetime.
Level of risk exposed if this best practice is not established: Medium
