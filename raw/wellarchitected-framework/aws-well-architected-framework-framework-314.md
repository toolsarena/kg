# AWS Well-Architected Framework Framework

Desired outcome: You encrypt network traffic between your resources and the internet to mitigate
unauthorized access to the data. You encrypt network traffic within your internal AWS environment
according to your security requirements. You encrypt data in transit using secure TLS protocols and
cipher suites.
