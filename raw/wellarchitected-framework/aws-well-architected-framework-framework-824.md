# AWS Well-Architected Framework Framework

• You do not document responsibilities.
• Fragmented scripts exist on isolated operator workstations. Only a few individuals know how to
use them or informally refer to them as team knowledge.
• A legacy process is due for update, but no one knows who owns the process, and the original
author is no longer part of the organization.
• Processes and scripts can't be discovered, and they are not readily available when required (for
example, in response to an incident).
